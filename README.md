## 🖥️ Project
Projeto Web Responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Este projeto foi desenvolvido durante o NLW da Rocketseat utilizando as seguintes tecnologias:

- HTML
- CSS
- Git e Github
